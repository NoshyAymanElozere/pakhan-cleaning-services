Abrechnung Von Transportaufträgen

Rechnung schreiben

1-Geben Sie den vollständigen Namen und die Anschrift Ihres Unternehmens an.

2-Fügen Sie Ihre Umsatzsteuer-Identifikationsnummer (USt-ID) oder Ihre Steuernummer hinzu

Unsere vollständige Firmenanschrift

1- Geben Sie die Firmenanschrift von TransCito an, an die die Rechnung gestellt wird.

2- Tragen Sie das Datum ein, an dem die Rechnung erstellt wurde

3- .Vergeben Sie eine eindeutige Rechnungsnummer.

4- Beschreiben Sie die erbrachte Leistung, z. B. „Transport von 51063 Köln nach 70435 Stuttgart

5- Geben Sie das Datum an, an dem die Leistung erbracht wurde (Tag der Zustellung)

6- Listen Sie den Nettobetrag der erbrachten Leistung auf

7- Weisen Sie die Umsatzsteuer (USt.) separat aus, es sei denn, Sie sind ein Kleinunternehmer gemäß §19 UStG. In diesem Fall dürfen Sie keine USt. in Rechnung stellen.

8- Geben Sie Ihre IBAN an, damit die Zahlung erfolgen kann.

9- Vergessen Sie nicht, die Auftragsnummer von TransCito auf der Rechnung zu vermerken. Dies ermöglicht eine schnelle Zuordnung der Rechnung. Wenn alle diese Punkte auf Ihrer Rechnung enthalten sind, wird eine korrekte und zügige Abrechnung gewährleistet.
